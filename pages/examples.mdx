# Examples

Here are some examples of how to use murphy.js in your projects.

## Basic Animations

These examples show the four basic animation directions available in murphy.js:

```html
<p data-murphy="bottom-to-top">Bottom to top</p>
<p data-murphy="top-to-bottom">Top to bottom</p>
<p data-murphy="left-to-right">Left to right</p>
<p data-murphy="right-to-left">Right to left</p>
```

## Custom Configuration

You can customize each animation with different attributes:

```html
<!-- Custom distance -->
<div data-murphy="bottom-to-top" data-murphy-element-distance="100">
  Moves 100px during animation
</div>

<!-- Custom duration -->
<div data-murphy="left-to-right" data-murphy-animation-duration="1000">
  Takes 1 second to animate
</div>

<!-- Custom easing -->
<div data-murphy="top-to-bottom" data-murphy-ease="cubic-bezier(0.4, 0, 0.2, 1)">
  Custom easing function
</div>

<!-- Custom threshold -->
<div data-murphy="right-to-left" data-murphy-element-threshold="0.5">
  Triggers when 50% visible
</div>
```
